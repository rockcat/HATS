Personality Project

The goal of personality project is to create AI agents with photo realistic avatars with distinct personalities as UI for AI processes.

They will have the following characteristics:
- Visual appearence for chat sessions this will include lip synced speech rendering through the use of prerendered frames 
- Personality traits - each AI agent avatar will ean toward one of De Bono's siz hats in personality
- Teamwork - all agents will desire to achieve a successful team result through working together
- Background working - agents will work independently until they need help and then ask other agents ot the user for a discussion
- Agents will arrange team calls when required

The shared personality core system will provide the following capabilites for agents:
- avatar creation - the user will describe a visual appearance, ie 'middle aged indian man in red shirt' and a photorealistic base image will be generate. Then a set of related images for lip syncing will be created.
- virtual webcam and audio to allow joining of team calls
- lipsync service to allow efficient text to lipsynced rendering of frames
